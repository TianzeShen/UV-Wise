1.0 Epic - Track UV Levels
As a young adult, I want to be able to know the UV levels so that I can prevent skin cancer.
US1.1  As a young adult spending time outdoors, I want to receive real-time, localised UV level alerts so that I know exactly when the radiation becomes dangerous and can take immediate action to protect my skin
Description : Must Have. Call API to view the UV index based on location. Colour accordingly to level. To design a feature that doesn't just show a number (e.g., UV 8), but translates that data into a 'Human Language' alert (e.g., 'Your skin will start damaging in 12 minutes—find shade now')
Benefit: Transforms an invisible threat (UV radiation) into a tangible metric. A young adult might not feel the sun's intensity on a cool, windy day, but seeing a "High" UV rating prompts them to apply sunscreen or seek shade. Relatable language. 

Acceptance criteria : 
Location detection
UV API retrieval
Color-coded UV display
Human readable alert

Epic 2.0 Raising awareness
As a young adult, I want to access tools that translate complex Australian UV data into relatable so that I can empower myself and my social circle to act as sun-safety influencers.

US2.1 As a young adult, I want to understand about Australia UV impacts and skin myths so that I can raise awareness to my friends. 

Description : Must Have. To show 2 visualisations of the impacts of skin cancer and the trend of heat in Australia. Database.
Benefits: Instead of telling young adults what to do, to show them visually on a graph the impacts of UV. When they understand the 'why,' they stop following orders and start making smart choices for themselves. By "raising awareness," the user becomes a hub for their friend group. They shift the responsibility from the individual to the group, making reapplication a collective social habit rather than a chore. 

Acceptance criteria : 
Educational Dashboard
Skin Cancer Statistics
UV Trend Visualization
Myth vs Fact Information
